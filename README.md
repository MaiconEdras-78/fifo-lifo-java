# 📚 Fila (FIFO) e Pilha (LIFO) em Java

Projeto desenvolvido para praticar estruturas de dados básicas.

## 🔹 Conceitos
- FIFO (First In, First Out)
- LIFO (Last In, First Out)

## 🛠 Tecnologias
- Java

## 📌 Funcionalidades
- Inserir elementos
- Remover elementos
- Visualizar estrutura

## ▶️ Como executar
1. Clone o repositório
2. Compile os arquivos Java
3. Execute a classe Main

## 🎯 Objetivo
Praticar lógica de programação e estruturas de dados
